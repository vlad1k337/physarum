# Physarum
Physarum Polycephalum algorithm implementation in C++17, using OpenGL 4.6

*Based on [this](https://uwe-repository.worktribe.com/output/980579) precious research paper*

## Quick start

Requires GLFW3 as a dependency!

### To build

Just run:

```console
$ make -j4
$ make clean
```
